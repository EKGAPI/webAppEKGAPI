ABOUT

Kardia.io aims to simplify biometrics for developers by providing a kit to streamline their own applications that leverage ECG data transmitted by the Kardiak wearable (displayed above).

HOW IT WORKS

Using our Kardia wearable provided, ECG data can be streamed via BLE from a user to their device (e.g. a mobile phone or game console) running our ECG analysis app. This data is then passed through our network, analyzed by a custom script, and passed back to the app for display. The app records any abnormal events that are reported by the server so the user can review them later.
